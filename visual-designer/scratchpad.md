2026-06-11 — SeoCta.tsx + seo-home.css (designer caught — RULE 14 violation in MY build) — the close-band heading "Stop doing the busywork. Start reviewing it." wrapped MID-SENTENCE ("busywork. Fix: render multi-sentence headings/short marketing lines split at full stops via `{text.split(/(?<=\.)\s+/).map(... <br> ...)}` so each sentence is its own line (and still wraps INTERNALLY if a line is too narrow — mobile-safe); switched `balance`→`pretty`, dropped the tight max-width. Applied to the close heading AND its sub. **GATE 8 MISS — I shipped a rule-14 violation in the new SEO build.** LESSON (reinforce rule 14 + promote): for ANY multi-sentence heading/short marketing line, DON'T use `text-wrap: balance` (it splits across the full stop) — break at the full stop (sentence-per-line) and let each sentence wrap internally.
